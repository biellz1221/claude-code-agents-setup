# Trend Scanner — Digital Marketing Intelligence

## Objective
Research and compile current trends, strategies and opportunities in digital marketing for **$ARGUMENTS**.

---

## Research Framework

### 1. Platform Trends (what's changing RIGHT NOW)

**Meta (Instagram + Facebook):**
```
Search:
  "instagram algorithm 2025 2026 changes"
  "facebook ads trends [current year]"
  "instagram reels best practices [current year]"
  "meta ads CPM trends"
  "instagram engagement rate benchmark"

Key things to track:
  - Algorithm changes (what content is being prioritized?)
  - New ad formats and placements
  - CPM/CPC trends (getting more or less expensive?)
  - Organic reach trends (increasing or dying?)
  - New features (Threads, Broadcast Channels, etc.)
```

**Google:**
```
Search:
  "google ads trends [current year]"
  "SEO trends [current year]"
  "google algorithm update [current year]"
  "AI overviews impact on SEO"
  "youtube shorts ads performance"

Key things to track:
  - Search behavior changes (AI overviews, zero-click searches)
  - Google Ads automation updates (Performance Max, etc.)
  - YouTube Shorts growth and monetization
  - Local SEO changes
  - Core Web Vitals updates
```

**TikTok:**
```
Search:
  "tiktok marketing trends [current year]"
  "tiktok ads performance benchmark"
  "tiktok shop [region]"
  "tiktok algorithm [current year]"

Key things to track:
  - TikTok Shop expansion
  - Search behavior on TikTok (replacing Google for Gen Z)
  - Ad format changes
  - Content trends (what formats are working?)
```

**Emerging Platforms:**
```
Search:
  "new social media platforms [current year]"
  "emerging marketing channels"
  "alternative to [platform] marketing"

Look for:
  - Threads / Bluesky growth
  - Newsletter platforms (Substack, Beehiiv)
  - Community platforms (Discord, Circle, Skool)
  - AI-native platforms
```

```
  - Short-form vs long-form performance

  - Interactive content (quizzes, calculators, tools)

```

**Paid Media Strategies:**
```
Search:
  "paid advertising trends [current year]"
  "facebook ads strategy [current year]"
  "google ads automation strategy"

Track:
  - Creative volume and testing velocity
  - Broad targeting vs detailed targeting
  - AI-driven optimization (Advantage+, Performance Max)

```

**Growth Strategies:**
```
Search:
  "SaaS growth strategy [current year]"

  "community-led growth"

Track:
  - PLG (Product-Led Growth) evolution
  - Community-led growth (building audiences)
  - AI-powered growth tactics
  - Partnership/integration marketing
  - Reverse trial models
```

## Analysis Framework

For each trend found, evaluate:

### Relevance Score (1-5)
How relevant is this for a solo dev building SaaS products?

### Actionability Score (1-5)
Can this be acted on immediately without a team or big budget?

### Timing (where in the adoption curve)
```
EARLY MOVER  -> Jump in NOW, first-mover advantage
GROWING      -> Still time, but competition increasing
MAINSTREAM   -> Table stakes, must do it or fall behind
DECLINING    -> Don't invest, find alternatives
```

### Effort/Impact Matrix
```
          LOW EFFORT     HIGH EFFORT
         +-------------+-------------+
HIGH     | * DO NOW    | PLAN FOR    |
IMPACT   |             |             |
         +-------------+-------------+
LOW      | AUTOMATE    | SKIP        |
IMPACT   |             |             |
         +-------------+-------------+
```

## Rules
- Use CURRENT data only — search with current year
- Every trend must include a "so what?" for a solo dev
- Prioritize ACTIONABLE trends over interesting-but-abstract ones
- Include regional market context (not just US-centric trends)
- Be honest about declining trends (save time by NOT doing them)
- Include specific tools and resources (not just concepts)
- Anti-trends are as valuable as trends (knowing what to stop)
- Every insight must pass the "can a solo dev do this?" test

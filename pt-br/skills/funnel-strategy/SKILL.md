# Funnel Strategy — Framework Russell Brunson

## Core Philosophy

"You're one funnel away."

Every successful product needs a SYSTEM that turns strangers into customers, not just a landing page with a "Buy Now" button. The funnel is that system.

Russell Brunson's key insight: **you don't sell a product, you sell a VALUE LADDER** — a journey where each step delivers more value at a higher price, and each step naturally leads to the next.

---

## The Value Ladder

```
                                    $$$$$
                                   ┌──────┐
                              HIGH │ Done  │ ← You do everything
                            VALUE  │ For   │    for the customer
                                   │ You   │
                                   ├──────┤
                                   │ Done  │ ← You help them
                              MED  │ With  │    do it
                            VALUE  │ You   │
                                   ├──────┤
                                   │ DIY   │ ← They do it with
                              LOW  │Course │    your content
                            VALUE  │/Tool  │
                                   ├──────┤
                              FREE │Lead   │ ← They get a taste
                                   │Magnet │    of your value
                                   └──────┘
                              $0    $$     $$$
```

### For SaaS Products (adapted):
```
FREE     → Free tier / Free trial / Lead magnet (PDF, tool, template)
$9-29    → Starter plan (core features, limited usage)
$49-99   → Pro plan (full features, priority support)
$199-499 → Team/Business (multi-user, integrations, SLA)
$1000+   → Enterprise / Custom / Done-for-you setup
```

### The Key Rule
**Never try to sell the expensive thing first.**
Give value at the bottom → Build trust → They naturally ascend.

## Funnel Types

### 1. Lead Magnet Funnel (TOFU — Free → Email List)
```
Traffic → Lead Magnet Page → Thank You Page → Email Sequence → Offer

Best for: Building an audience, SaaS waitlist, content marketing
Lead magnet ideas for SaaS:
  - Free tool / calculator / generator
  - PDF guide: "O Guia Definitivo de [topic]"
  - Template: spreadsheet, Notion, code snippet
  - Mini course (3-5 emails)
  - Free audit / assessment
  - Checklist: "[N] passos pra [result]"

Page structure:
  Headline: "[Get specific result] — Grátis"
  Subheadline: what they'll learn/get
  Bullet points: 3-5 specific benefits
  Form: Name + Email only (low friction)
  Social proof: "Já baixaram: 2.500+ devs"
```

### 2. Tripwire Funnel (TOFU → Low-ticket → Upsell)
```
Traffic → Low-ticket Offer ($7-27) → Order Bump → Upsell → Thank You

Best for: Acquiring customers at break-even, building buyer list
The goal: NOT profit. Cover ad costs while building a list of BUYERS.
Buyers are 10x more valuable than freebie seekers.

Structure:
  Page 1: Irresistible offer at break-even price
  Order bump: Related add-on for +30-50% of price
  Upsell 1: The main product at discount
  Upsell 2: Premium tier / annual plan
  Downsell: Lighter version if they reject upsell

For SaaS:
  Tripwire: Lifetime access to a micro-tool ($17-27)
  Order bump: Template pack (+$12)
  Upsell: Annual plan at 40% off ($X/year vs monthly)
```

### 3. Webinar Funnel (MOFU → Education → Sale)
```
Traffic → Registration Page → Webinar → Offer → Follow-up Emails

Best for: Higher-ticket SaaS, complex products that need explanation
The webinar is NOT a sales pitch — it's a teaching session that naturally
leads to the product as the logical next step.

Structure:
  Registration page:
    "Masterclass gratuita: Como [result] em [time] sem [objection]"
    Date/time + scarcity ("vagas limitadas")
    3 bullet points of what they'll learn

  Webinar content (60-90 min):
    0-5min: Hook + credibility
    5-15min: Story (the origin, the problem you solved)
    15-45min: 3 secrets/strategies (pure value)
    45-60min: The stack (present your offer layer by layer)
    60-75min: Q&A + urgency (bonus expires, price increases)

  Follow-up sequence (7 days):
    Day 0: Replay link + recap
    Day 1: Case study / testimonial
    Day 2: FAQ / objection handling
    Day 3: "Price goes up tomorrow"
    Day 4-7: Last chance + downsell alternative
```

### 4. Challenge Funnel (5-7 Day Challenge → Sale)
```
Traffic → Challenge Signup → Daily Content → Offer on Day 5-7

Best for: Products that benefit from guided onboarding, community building
The challenge USES the product, so by the end they're already activated.

Structure:
  Signup page:
    "Desafio de 5 dias: [Specific result] usando [Product]"
    Free signup (or $7-27 to filter serious people)

  Daily emails + tasks:
    Day 1: Setup + quick win (5-minute task)
    Day 2: Core feature deep-dive
    Day 3: Advanced technique
    Day 4: Integration / workflow
    Day 5: Results review + offer

For SaaS: The challenge IS the onboarding.
By Day 5 they've used the product, gotten results, and upgrading is natural.
```

### 5. Application Funnel (High-ticket — $500+)
```
Traffic → Application Page → Call Booking → Sales Call → Close

Best for: Enterprise SaaS, done-for-you services, consulting
The application QUALIFIES leads before you invest time in a call.

Structure:
  Application page:
    "Aplique para [result]"
    Case studies + results
    Application form (5-10 qualifying questions)
    "Vagas limitadas" (scarcity — real or perceived)

  Call structure (30-45 min):
    5min: Rapport + understand their situation
    10min: Deep dive into their problem
    10min: Present the solution (your offer)
    10min: Handle objections
    5min: Close or next steps
```

## Email Sequences (post-funnel)

### Welcome Sequence (everyone who enters the funnel)
```
Email 1 (immediate): Deliver the lead magnet + set expectations
Email 2 (Day 1): Your story + why you built this
Email 3 (Day 2): Quick win / tutorial using the product
Email 4 (Day 3): Case study / social proof
Email 5 (Day 5): Common mistake + how to avoid
Email 6 (Day 7): Soft offer (trial, demo, or paid plan)
Email 7 (Day 10): Direct offer + urgency
```

### Abandoned Cart / Trial Expiring
```
Email 1 (1 hour): "Esqueceu algo?" + remind benefits
Email 2 (24 hours): Social proof / case study
Email 3 (48 hours): Handle top objection
Email 4 (72 hours): Final reminder + bonus/discount
```

## Process

1. **Analyze the product** — understand the value ladder potential
2. **Identify the best funnel type** — based on price point, audience, and complexity
3. **Design the funnel** — page by page, email by email, step by step
4. **Write the offer stack** — what they get, bonuses, guarantee, price
5. **Plan traffic strategy** — where to send people into the funnel
6. **Define success metrics** — conversion rates at each step

## Output

Generate a complete funnel blueprint as a markdown document:
- Visual funnel diagram (ASCII)
- Page-by-page breakdown with copy guidelines
- Email sequence with subject lines and content outline
- Offer stack with pricing strategy
- Traffic recommendation
- KPIs per funnel step
- Tech stack recommendation (what to build it with)

---

## Rules
- Every funnel step must deliver VALUE (not just collect emails)
- The free thing must be genuinely useful (bad lead magnets = bad leads)
- Scarcity must be REAL — fake urgency destroys trust
- Each step should naturally lead to the next (no hard pivots)
- Test the simplest version first, optimize later
- For a solo dev: build the MVP funnel in a weekend, optimize over months
- Always include a money-back guarantee on paid offers (reduces friction dramatically)
- Price anchoring: show the value first, then the price seems small
